# Habit Tracker

This Python project allows users to track habits visually using the [Pixela API](https://pixe.la/). The application enables users to log daily activities, update entries, and delete them if necessary.

---

## Features
- **User Account Creation**: Create a Pixela user account programmatically.
- **Graph Creation**: Set up a habit graph to track your progress.
- **Daily Logging**: Log data for a specific date.
- **Entry Updates**: Update graph entries (e.g., change data or graph colors).
- **Entry Deletion**: Remove an entry from the graph.

---

## Requirements
- Python 3.x
- API access to [Pixela](https://pixe.la/).

---

## Configuration
- TOKEN = "YOUR_UNIQUE_TOKEN"  # Replace with your Pixela token
- USERNAME = "YOUR_USERNAME"  # Replace with your Pixela username
- ID = "graph1"  # Replace with your preferred graph ID
